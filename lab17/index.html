Step 1: Understand Problem

  • Set up game initial board and player
  • draw the board
  • Figure out the word player will be guessing
  • Get guess from player
  • check if letter is in the word or not
  • game loop
  • check if player guessed the word condition

Step 2: Pseudocode High Level Approach

  • Set up initial game
    • Create game board
    • Create Player
  • Start game loop
    • Render game board
    • Render random word that will be guessed
    • Ask for and validate guess from player
    • If letter in the word is guessed
      • Display letter in corresponding area
      • keep looping
    • Else
      • Start placing the man on the board
      • Let player guess different letter
      • Keep looping
    • If whole word is guessed before certain amount of guesses
      • Display proper victory message
      • Stop looping
    • Else
      • Display game over message
      • Stop looping

Step 3: Assign Actions to Objects

  • Set up initial game [Hangman]
    • Create game board [Board]
    • Create Player [Player]
  • Start game loop [Hangman]
    • Render game board [Board]
    • Render random word that will be guessed [Word]
    • Ask for and validate guess from player [Player]
    • If letter in the word is guessed [Word]
      • Display letter in corresponding area [Board]
      • keep looping [Hangman]
    • Else
      • Start placing the man on the board [Man]
      • Let player guess different letter [Player]
      • Keep looping [Hangman]
    • If whole word is guessed before certain amount of guesses [Word]
      • Display proper victory message
      • Stop looping [Hangman]
    • Else
      • Display game over message
      • Stop looping [Hangman]
